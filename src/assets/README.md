# Assets reales

La app usa directamente estos archivos locales ya disponibles:

- `src/assets/fragments/ChatGPT Image 24 abr 2026, 08_26_23 p.m..png`
- `src/assets/fragments/WhatsApp Image 2026-04-24 at 19.45.46.jpeg`

No hay carga manual desde la UI. La captura de WhatsApp se desfragmenta automaticamente en 12 cortes desde Canvas al abrir la pagina y se pueden descargar como JPG.

Los placeholders SVG generados fueron retirados para no reemplazar ni reinterpretar las imagenes originales.
